# Live-waterstanden
Dit Excelbestand geeft live inzicht in waterstanden en aflaaddieptes voor de binnenvaart. Het systeem update automatisch elke 30 minuten en berekent direct de beschikbare diepgang op basis van uw vuistregels. Het dashboard toont per traject de kritische punten, zodat u snel en efficiënt kunt plannen.
